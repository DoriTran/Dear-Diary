# Dear Diary

Dear Diary is a simple **note-taking app** with a **chat-style layout**, so your notes feel like an ongoing conversation instead of static pages.

You can write, review, and organize your thoughts in a conversational flow, similar to a messaging app.

## Getting Started

### Prerequisites

- **Node.js** (LTS version recommended)
- **npm** or **pnpm** or **yarn**

### Install dependencies

Using npm:

```bash
npm install
```

Or with pnpm:

```bash
pnpm install
```

Or with yarn:

```bash
yarn install
```

### Run the app in development

```bash
npm run dev
```

Then open the printed local URL (usually `http://localhost:5173`) in your browser to use the app.
